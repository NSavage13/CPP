Directions

This week the assignments are a little complex. You will do two classes and inheritances. You will select two from the first group. These are the classes that you will build. Then in the second section, you will perform the related inheritance exercises, for the classes you built. 

For clarity, if you choose the sandwich class (#3) in the first group, then you will choose the Hoagie (#3) in the second group.

Classes

Choose and complete two (2) of the following class exercises

For each of these classes, you will perform the following
•    Create parameterized and default, constructors.
•    Create the setters and getters for each variable
•    Create a to_string function
•    Instantiate a variable of the class in the main
•    Demonstrate that all methods work
•    Create an array of the class with instantiated variables

  1. Create a class that holds a car. Include fields of make, model, year, miles, gallons, and MPG. Create a method that calculates the distance the car can travel on the gas, and a method that determines cost per mile, assuming that gas costs $3.09 per gallon.
    • Cost per mile = 3.09 / (mpg * gallons)
    • Distance = mpg * gallons
  2. Create a class that holds a patient’s medical records. The class must hold fname, lname, height, weight. Create a method that calculates the BMI (body mass index).
  3. Create a class called Sandwich. The class will calculate the cost of a sandwich, based on the items you choose. Your class will hold the bread type, meat type, cheese type, vegetables, condiment, and price. Create a method that calculates the price of the sandwich based on the bread type, meat type, and cheese type.
    • For clarity, different breads cost a different amount, likewise meats and cheeses each can cost a different amount. Create as many additional constructors as makes sense. Create whatever methods you think are appropriate.
  4. Create a class that holds the information for a monster from a game. Variables should include at least name, damage done, health, and attack type. Include methods that allow the monster to take damage and do damage.

Inheritance

For each of these classes you will perform the following
•    Create parameterized and default constructors that reference the parent constructors
•    Create the setters and getters for each variable in the inherited class
•    Create a to_string function that uses the parent’s to_string method
•    Instantiate a variable of the class in the main
•    Demonstrate that all methods work
•    Create an array of the class with instantiated variables

  1. Create a class called Hybrid that inherits from car. Add the following variables: batteryCapacity, and batteryChargePercent. Create a method that accepts a parameter that will ‘charge’ the battery based on the number passed in. Create a method that calculates the ‘range’ of the hybrid. For simplicity, assume that the car can go 10 miles for every % charge in the battery.
  2. Create a class called CardiacPatient that inherits from Patient. The additional class variables are the blood pressure systolic and diastolic numbers. Create a method that determines the patient’s at-risk levels based on the blood pressure (lookup the levels on Google).
  3. Create a class called Hoagie that inherits from Sandwich. Add two class variables called length in feet and baked. Override the cost calculation method as follows.
    • Total price = price*feet. If the hoagie is baked, add $.75 per foot to the price.
  4. Create a class called boss that inherits from Monster. Add a shield and a super attack. Change the setter and constructor for health to add 50% more health and damage to the monster.  Override the damage done to the monster to first take the damage from the shield. Only once the shield is depleted, does damage come from the health. Add a second method to allow the boss to ‘attack’ with it’s super attack.
